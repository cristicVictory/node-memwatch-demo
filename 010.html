Finding Leaks
     
     Now that we have a leak detector, we want a leak identifier. By comparing
     consecutive heap snapshots, we can produce a heap diff.
     
     * var hd = new memwatch.HeapDiff();
       // do stuff ...
       var diff = hd.end()
